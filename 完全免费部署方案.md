## 📋 Oracle Cloud 详细步骤

### 第一步：注册账号（10分钟）

1. **访问**：https://www.oracle.com/cloud/free/
2. 点击"Start for free"
3. 填写信息：
   - 国家/地区：中国
   - 邮箱：你的邮箱
   - 名字：Zhang
   - 姓氏：Guiquan
4. 验证邮箱
5. 填写详细信息：
   - 地址：随便填（英文）
   - 电话：你的手机号
6. **信用卡验证**：
   - 需要Visa/Mastercard信用卡
   - 会扣1美元验证（之后退回）
   - 不会自动扣费
7. 等待审核（通常几分钟）

**没有信用卡？** 跳到方案二（Render）

### 第二步：创建虚拟机（5分钟）

1. 登录Oracle Cloud控制台
2. 点击"Create a VM instance"
3. 配置：
   - Name: cloudnote
   - Image: Ubuntu 22.04（推荐）
   - Shape: VM.Standard.A1.Flex（ARM，免费）
   - OCPU: 1
   - Memory: 1GB
4. 网络配置：
   - 使用默认VCN
   - 分配公网IP：是
5. SSH密钥：
   - 下载私钥（保存好！）
6. 点击"Create"
7. 等待创建（约2分钟）
8. 获得公网IP：123.456.789.123

### 第三步：配置防火墙（5分钟）

1. 在虚拟机详情页
2. 点击"Subnet"
3. 点击"Default Security List"
4. 点击"Add Ingress Rules"
5. 添加规则：
   - Source CIDR: 0.0.0.0/0
   - Destination Port: 8080（后端）
   - 点击"Add Ingress Rules"
6. 再添加：
   - Destination Port: 80（HTTP）
   - Destination Port: 443（HTTPS）

### 第四步：连接服务器（5分钟）

**Windows用户**：
1. 下载Xshell：https://www.xshell.com/zh/free-for-home-school/
2. 新建会话：
   - Host: 你的公网IP
   - Protocol: SSH
   - Port: 22
3. 用户身份验证：
   - Method: Public Key
   - User Name: ubuntu
   - User Key: 导入下载的私钥
4. 连接

**或使用Oracle Cloud Shell**（网页终端）

### 第五步：安装环境（10分钟）

连接成功后，执行以下命令：

```bash
# 更新系统
sudo apt update
sudo apt upgrade -y

# 安装Java
sudo apt install openjdk-17-jdk -y
java -version

# 安装MySQL
sudo apt install mysql-server -y
sudo systemctl start mysql
sudo systemctl enable mysql

# 配置MySQL
sudo mysql
```

在MySQL中执行：
```sql
-- 创建数据库
CREATE DATABASE cloudnote CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户
CREATE USER 'cloudnote'@'localhost' IDENTIFIED BY '123456';
GRANT ALL PRIVILEGES ON cloudnote.* TO 'cloudnote'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

```bash
# 安装Nginx
sudo apt install nginx -y
sudo systemctl start nginx
sudo systemctl enable nginx

# 开放防火墙（Ubuntu内部）
sudo ufw allow 8080
sudo ufw allow 80
sudo ufw allow 443
sudo ufw enable
```

### 第六步：上传文件（10分钟）

**方式一：使用SCP（推荐）**

在你的电脑上：
```bash
# 上传JAR包
scp -i 私钥文件 cloudnote-1.0.0.jar ubuntu@你的IP:/home/ubuntu/

# 上传数据库文件
scp -i 私钥文件 cloudnote.sql ubuntu@你的IP:/home/ubuntu/

# 上传前端文件（压缩后）
cd front_end/dist
tar -czf dist.tar.gz *
scp -i 私钥文件 dist.tar.gz ubuntu@你的IP:/home/ubuntu/
```

**方式二：使用WinSCP**
1. 下载WinSCP：https://winscp.net/
2. 新建连接
3. 使用私钥登录
4. 拖拽上传文件

### 第七步：部署后端（5分钟）

在服务器上：
```bash
# 创建目录
sudo mkdir -p /opt/cloudnote
sudo mv cloudnote-1.0.0.jar /opt/cloudnote/
cd /opt/cloudnote

# 导入数据库
mysql -u cloudnote -p123456 cloudnote < /home/ubuntu/cloudnote.sql

# 启动后端
nohup java -jar cloudnote-1.0.0.jar > app.log 2>&1 &

# 查看日志
tail -f app.log
```

看到"Started CloudNoteApplication"表示成功！

### 第八步：部署前端（5分钟）

```bash
# 解压前端文件
sudo mkdir -p /var/www/cloudnote
cd /var/www/cloudnote
sudo tar -xzf /home/ubuntu/dist.tar.gz

# 配置Nginx
sudo nano /etc/nginx/sites-available/cloudnote
```

添加以下内容：
```nginx
server {
    listen 80;
    server_name _;

    # 前端
    location / {
        root /var/www/cloudnote;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端API
    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

保存（Ctrl+O，Enter，Ctrl+X）

```bash
# 启用配置
sudo ln -s /etc/nginx/sites-available/cloudnote /etc/nginx/sites-enabled/
sudo rm /etc/nginx/sites-enabled/default
sudo nginx -t
sudo systemctl restart nginx
```

### 第九步：测试访问

访问：`http://你的公网IP`

应该看到登录页面！

## 🌟 方案二：Render（最简单，完全免费）

### 优点
- ✅ **完全免费**（永久）
- ✅ **最简单**（网页操作）
- ✅ **不需要信用卡**
- ✅ **自动HTTPS**
- ✅ **免费域名**

### 缺点
- ⚠️ 会休眠（15分钟无访问）
- ⚠️ 首次访问慢（需要唤醒）
- ⚠️ 配置低（512MB内存）

### 详细步骤

#### 1. 注册Render
- 访问：https://render.com
- 使用邮箱注册（不需要信用卡）

#### 2. 上传代码到GitHub
**必须用GitHub**，但很简单：
```bash
# 在项目目录
git init
git add .
git commit -m "Deploy to Render"

# 在GitHub创建仓库（网页操作）
# 然后：
git remote add origin https://github.com/你的用户名/cloudnote.git
git push -u origin master
```

#### 3. 部署后端
1. 在Render点击"New +" → "Web Service"
2. 连接GitHub仓库
3. 配置：
   - Name: cloudnote-backend
   - Environment: Java
   - Build Command: `mvn clean package -DskipTests`
   - Start Command: `java -jar target/cloudnote-1.0.0.jar`
   - Instance Type: **Free**
4. 点击"Create Web Service"
5. 等待部署（约5分钟）

#### 4. 添加数据库
使用免费MySQL服务：
- **Aiven**：https://aiven.io/free-mysql-database
- 注册并创建免费MySQL
- 获取连接信息

#### 5. 配置环境变量
在Render后端服务中：
1. 点击"Environment"
2. 添加变量：
   - `MYSQL_URL`: jdbc:mysql://你的数据库地址
   - `MYSQL_USER`: 用户名
   - `MYSQL_PASSWORD`: 密码
3. 保存并重新部署

#### 6. 部署前端
1. 点击"New +" → "Static Site"
2. 连接GitHub仓库
3. 配置：
   - Name: cloudnote-frontend
   - Root Directory: front_end
   - Build Command: `npm run build`
   - Publish Directory: `dist`
4. 点击"Create Static Site"

#### 7. 获取域名
- 后端：`https://cloudnote-backend.onrender.com`
- 前端：`https://cloudnote-frontend.onrender.com`
