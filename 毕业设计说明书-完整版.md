## 一、项目概述

### 1.1 项目背景

随着信息时代的到来，知识管理成为提升学习和工作效率的关键。传统纸质笔记存在检索困难、易丢失、无法跨设备访问等问题。本项目开发了一套基于Spring Boot和Vue 3的智能AI云笔记管理系统，集成了AI学习助手功能，为用户提供智能化的知识管理解决方案。

### 1.2 项目目标

- 实现完整的云笔记管理功能（创建、编辑、分类、标签、搜索）

- 采用前后端分离架构，提升系统可维护性
- 实现安全的用户认证与权限控制
- 提供友好的用户界面和流畅的交互体验

## 二、系统功能设计

### 2.1 功能模块架构

系统主要包含以下六大功能模块：

1. **用户管理模块** - 用户注册、登录、个人信息管理、密码修改
2. **笔记管理模块** - 创建、编辑、删除、查询、搜索笔记
3. **分类管理模块** - 创建、编辑、删除分类，按分类筛选笔记
4. **标签管理模块** - 创建、编辑、删除标签，按标签筛选笔记
5. **AI智能助手模块** ⭐ - 智能问答、学习辅助、上下文对话
6. **回收站模块** - 查看已删除笔记、恢复笔记、彻底删除

### 2.2 核心功能详解

#### 2.2.1 用户管理模块

**功能描述：**
- 用户注册：通过用户名、密码、邮箱完成注册，系统进行参数校验
- 用户登录：验证用户名和密码，登录成功后生成JWT Token
- 个人信息管理：查看和修改昵称、邮箱、头像等个人信息
- 密码修改：验证旧密码后修改新密码，密码采用BCrypt加密

**技术实现：**
- Spring Validation进行参数校验（用户名3-20字符，密码6-20字符）
- BCrypt算法加密密码，安全强度10轮
- JWT Token有效期7天，包含用户ID和用户名
- 拦截器统一处理Token验证，未登录用户自动跳转登录页

#### 2.2.2 笔记管理模块

**功能描述：**
- 创建笔记：输入标题、内容，选择分类，添加标签
- 编辑笔记：修改笔记内容，系统自动更新时间戳
- 删除笔记：逻辑删除，笔记移入回收站而非物理删除
- 列表查询：分页展示笔记列表，按更新时间倒序排列
- 全文搜索：在标题和内容中模糊匹配关键词

**技术实现：**
- MyBatis Plus提供BaseMapper接口，无需编写基础CRUD代码
- 分页插件自动拼接LIMIT语句，实现物理分页
- LIKE语句实现全文搜索（标题 LIKE '%关键词%' OR 内容 LIKE '%关键词%'）
- deleted字段标记删除状态（0=正常，1=已删除）

#### 2.2.3 分类管理模块

**功能描述：**
- 创建分类：用户可创建个性化分类（如"工作笔记"、"学习笔记"）
- 编辑分类：修改分类名称
- 删除分类：删除前检查是否有关联笔记，有则提示先处理笔记
- 分类筛选：点击分类可查看该分类下的所有笔记

**技术实现：**
- 数据库设置user_id和name的联合唯一索引，确保同一用户下分类名称唯一
- 外键关联笔记表的category_id字段
- 删除前通过COUNT查询统计分类下笔记数量

#### 2.2.4 标签管理模块

**功能描述：**
- 创建标签：在创建或编辑笔记时添加标签，多个标签用逗号分隔
- 标签列表：自动提取用户使用过的所有标签
- 标签筛选：点击标签可筛选包含该标签的所有笔记
- 删除标签：删除不再使用的标签

**技术实现：**
- 标签以字符串形式存储在笔记表的tags字段（如"Java,Spring,后端"）
- 前端通过split(',')分割标签字符串

## 四、关键技术实现

### 4.1 JWT身份认证

**实现原理：**
1. 用户登录成功后，服务器生成JWT Token
2. Token包含用户ID、用户名、签发时间、过期时间
3. 客户端将Token存储在localStorage
4. 后续请求在Header中携带：`Authorization: Bearer {token}`
5. 服务器通过拦截器验证Token有效性

**核心代码：**

```java
// JWT工具类 - 生成Token
public String generateToken(Integer userId, String username) {
    Date now = new Date();
    Date expiryDate = new Date(now.getTime() + 7 * 24 * 60 * 60 * 1000);
    
    return Jwts.builder()
            .setSubject(userId.toString())
            .claim("username", username)
            .setIssuedAt(now)
            .setExpiration(expiryDate)
            .signWith(SignatureAlgorithm.HS512, secret)
            .compact();
}

// JWT拦截器 - 验证Token
@Override
public boolean preHandle(HttpServletRequest request, 
                        HttpServletResponse response, 
                        Object handler) {
    String token = request.getHeader("Authorization");
    if (token != null && token.startsWith("Bearer ")) {
        token = token.substring(7);
        if (jwtUtil.validateToken(token)) {
            Integer userId = jwtUtil.getUserIdFromToken(token);
            UserContext.setUserId(userId); // 存入ThreadLocal
            return true;
        }
    }
    throw new UnauthorizedException("未授权，请先登录");
}
```

### 4.4 前端Axios请求封装

```javascript
// 请求拦截器 - 自动添加Token
request.interceptors.request.use(
  config => {
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  }
)

// 响应拦截器 - 统一错误处理
request.interceptors.response.use(
  response => {
    const res = response.data
    if (res.code !== 200) {
      ElMessage.error(res.message)
      return Promise.reject(new Error(res.message))
    }
    return res.data
  },
  error => {
    if (error.response && error.response.status === 401) {
      ElMessage.error('登录已过期，请重新登录')
      localStorage.removeItem('token')
      router.push('/login')
    }
    return Promise.reject(error)
  }
)
```

## 七、项目创新点

### 7.1 集成AI智能助手 ⭐⭐⭐（最大创新点）

这是本系统最大的创新点和亮点：

**1. 学习场景深度结合**
- 将AI助手与笔记系统结合，用户在学习过程中遇到问题可立即咨询
- 无需切换应用，提升学习效率
- 边学边问，边问边记，形成完整的学习闭环

**2. 多API支持策略**
- 提供3种免费AI API选择（Groq、Cohere、HuggingFace）
- 用户可根据速度、质量、额度自由切换
- 降低单一API依赖风险，提高系统可用性

**3. 零成本使用**
- 所有AI API均为免费方案
- Groq API每天14,400次请求，完全满足学习需求
- 降低用户使用门槛，人人都能用上AI助手

**4. 智能上下文管理**
- 保留最近5轮对话作为上下文
- AI能理解连续对话，提供更准确的回答
- 支持深入追问，像真人对话一样自然

**5. 友好的用户体验**
- 快速问题模板，一键发送
- Markdown渲染，代码高亮
- 实时打字动画，增强交互感
- 详细的API配置帮助文档

### 7.2 前后端完全分离

- 前端Vue 3独立部署，可部署到CDN
- 后端Spring Boot提供RESTful API
- 便于团队协作和功能扩展
- 前后端可独立升级维护

### 7.3 安全性设计

- JWT无状态认证，支持分布式部署
- BCrypt密码加密，安全强度高
- SQL注入防护（MyBatis Plus参数化查询）
- XSS攻击防护（前端输入转义）
- CORS跨域配置，限制访问来源

### 7.4 用户体验优化

- 响应式设计，适配PC、平板、手机
- 加载动画和过渡效果，提升视觉体验
- 友好的错误提示，帮助用户快速定位问题
- 快捷键支持（Enter发送，Shift+Enter换行）
