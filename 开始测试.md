## 🐛 遇到问题？

### 常见问题解决

#### 问题 1: 页面显示 404
```bash
# 解决方案: 清除浏览器缓存
localStorage.clear()
# 然后刷新页面
```

#### 问题 2: 接口返回 401
```bash
# 解决方案: 重新登录
1. 清除缓存: localStorage.clear()
2. 访问登录页
3. 重新登录
```

#### 问题 3: 接口返回 500
```bash
# 解决方案: 检查后端
1. 查看后端控制台错误日志
2. 确认数据库连接正常
3. 重启后端服务
```
