# ⚡ 快速测试步骤

## 🎯 5 分钟快速测试

### 1. 访问个人信息页面
```
1. 访问 http://localhost:5173
2. 登录: testuser / 123456
3. 点击右上角头像 → 个人信息
```

### 2. 测试修改用户名
```
1. 点击"编辑信息"
2. 修改用户名为 testuser2
3. 点击"保存"
4. 验证: 显示"信息更新成功"
```

### 3. 测试修改密码
```
1. 切换到"修改密码"标签页
2. 旧密码: 123456
3. 新密码: 654321
4. 确认密码: 654321
5. 点击"修改密码"
6. 验证: 3秒后跳转登录页
7. 使用新密码 654321 登录
```

### 4. 测试表单验证
```
1. 点击"编辑信息"
2. 清空用户名 → 验证错误提示
3. 输入无效邮箱 → 验证错误提示
4. 点击"取消" → 验证恢复原值
```

### 5. 测试账户安全
```
1. 切换到"账户安全"标签页
2. 点击"修改密码" → 验证跳转
3. 点击"修改邮箱" → 验证跳转
```

## ✅ 测试通过标准

- ✅ 所有页面正常显示
- ✅ 所有表单验证生效
- ✅ 所有操作成功提示
- ✅ 修改密码后能重新登录
- ✅ 响应式设计正常

## 🐛 如遇问题

1. 清除缓存: `localStorage.clear()`
2. 重启后端服务
3. 查看浏览器控制台错误
4. 查看后端控制台日志
