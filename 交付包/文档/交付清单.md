## 🚀 快速打包步骤

### 步骤1：打包后端
```bash
# 在项目根目录执行
mvn clean package -DskipTests

# 生成文件：target/cloudnote-1.0.0.jar
```

### 步骤2：打包前端
```bash
# 进入前端目录
cd front_end

# 安装依赖（如果没装过）
npm install

# 打包
npm run build

# 生成目录：front_end/dist/
```

### 步骤3：创建交付压缩包
```bash
# 创建交付目录
mkdir -p 交付包/源代码
mkdir -p 交付包/部署包
mkdir -p 交付包/文档

# 复制源代码（排除 node_modules 和 target）
xcopy /E /I /EXCLUDE:exclude.txt . 交付包\源代码

# 复制部署包
copy target\cloudnote-1.0.0.jar 交付包\部署包\
xcopy /E /I front_end\dist 交付包\部署包\front_end_dist

# 复制文档
copy *.md 交付包\文档\
copy CloudNote项目演示.html 交付包\文档\
copy src\main\resources\db\schema.sql 交付包\文档\

# 压缩
# 使用 7-Zip 或 WinRAR 压缩 "交付包" 文件夹
```

## 🎯 老师可能会问的问题

1. **项目架构是什么？**
   - 前后端分离，Spring Boot + Vue 3

2. **认证方式是什么？**
   - JWT 无状态认证

3. **遇到了什么技术难点？**
   - 标签系统重构（多对多关系）
   - MyBatis Plus 逻辑删除冲突
   - 图片上传优化（Base64 → 文件存储）

4. **如何保证数据安全？**
   - JWT Token 认证
   - 数据级权限控制
   - BCrypt 密码加密

5. **如何部署？**
   - 后端：java -jar cloudnote-1.0.0.jar
   - 前端：Nginx 部署 dist 目录
