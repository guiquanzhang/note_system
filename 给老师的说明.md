## 🚀 快速运行指南

### 环境要求
- **JDK**: 1.8+
- **MySQL**: 8.0+
- **Node.js**: 16.0+（仅开发环境需要）

### 运行步骤

#### 1. 准备数据库

```sql
-- 创建数据库
CREATE DATABASE cloudnote CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 导入数据（如果提供了 SQL 文件）
-- 使用 Navicat 或命令行导入 cloudnote-database.sql
```

**数据库配置**：
- 地址：`localhost:3306`
- 数据库名：`cloudnote`
- 用户名：`root`
- 密码：`123456`

如需修改，请编辑：`src/main/resources/application.yml`

---

#### 2. 运行后端

**方式一：运行 JAR 包（推荐）**

如果提供了 JAR 文件：
```bash
java -jar cloudnote-0.0.1-SNAPSHOT.jar
```

**方式二：使用 IDE 运行**

1. 用 IntelliJ IDEA 打开项目
2. 找到 `src/main/java/com/cloudnote/CloudNoteApplication.java`
3. 右键 → Run

**方式三：使用 Maven 打包运行**

```bash
# 打包
mvn clean package -DskipTests

# 运行
java -jar target/cloudnote-0.0.1-SNAPSHOT.jar
```

**验证后端启动**：
- 访问：http://localhost:8080
- 看到页面或 JSON 响应即表示成功

---

#### 3. 运行前端

**方式一：直接访问打包文件（推荐）**

1. 将 `dist` 目录部署到 Web 服务器（如 Nginx）
2. 或者用 VS Code 的 Live Server 插件打开 `dist/index.html`

**方式二：开发模式运行**

```bash
cd front_end
npm install
npm run dev
```

访问：http://localhost:5173

---

#### 4. 访问系统

- **前端地址**：http://localhost:5173（开发模式）或 http://localhost（生产模式）
- **后端地址**：http://localhost:8080
- **API 文档**：http://localhost:8080/doc.html

## 🎯 技术亮点

### 后端技术
- **Spring Boot 2.7.18** - 核心框架
- **MyBatis Plus 3.5.3.1** - ORM 框架
- **MySQL 8.0** - 数据库
- **JWT** - 身份认证
- **Swagger 3.0** - API 文档
- **BCrypt** - 密码加密

### 前端技术
- **Vue 3** - 渐进式框架
- **Element Plus** - UI 组件库
- **Pinia** - 状态管理
- **Vue Router** - 路由管理
- **Axios** - HTTP 客户端
- **Vite** - 构建工具

### 架构特点
- ✅ 前后端分离
- ✅ RESTful API 设计
- ✅ JWT 无状态认证
- ✅ 统一响应格式
- ✅ 全局异常处理
- ✅ 软删除机制

## 📁 项目结构

```
cloudnote/
├── front_end/              # 前端项目
│   ├── dist/               # 打包后的前端文件 ⭐
│   ├── src/                # 前端源代码
│   │   ├── api/            # API 接口
│   │   ├── components/     # 组件
│   │   ├── views/          # 页面
│   │   ├── router/         # 路由
│   │   ├── store/          # 状态管理
│   │   └── utils/          # 工具类
│   └── package.json        # 依赖配置
│
├── src/main/               # 后端源代码
│   ├── java/com/cloudnote/
│   │   ├── controller/     # 控制器
│   │   ├── service/        # 服务层
│   │   ├── mapper/         # 数据访问层
│   │   ├── entity/         # 实体类
│   │   ├── dto/            # 数据传输对象
│   │   ├── common/         # 通用类
│   │   ├── config/         # 配置类
│   │   ├── interceptor/    # 拦截器
│   │   └── exception/      # 异常处理
│   └── resources/
│       ├── application.yml # 配置文件
│       └── db/             # 数据库脚本
│
├── target/                 # 编译输出目录
│   └── cloudnote-0.0.1-SNAPSHOT.jar  # 可执行 JAR ⭐
│
├── uploads/                # 文件上传目录
├── pom.xml                 # Maven 配置
└── README.md               # 项目说明
```

### 支持的 AI 服务

   - 无需翻墙
   - GLM-4-Flash 模型免费
   - 中文能力强

3. **硅基流动**
   - 国内访问
   - 支持多种模型

### 使用方法

1. 登录系统后，点击左侧菜单"AI 助手"
2. 点击右上角"设置"按钮
3. 选择 AI 提供商
4. 输入 API Key（需要自己注册获取）
5. 保存设置后即可使用

**注意**：由于 API Key 是个人的，未在代码中硬编码。如需测试 AI 功能，请按照帮助文档获取免费 API Key。
