# 富文本编辑器使用技巧

## 基础功能

### 文本格式
- **加粗**：选中文字后点击 B 按钮，或按 Ctrl+B
- **斜体**：选中文字后点击 I 按钮，或按 Ctrl+I
- **下划线**：选中文字后点击 U 按钮，或按 Ctrl+U
- **删除线**：选中文字后点击 S 按钮

### 标题
1. 将光标放在要设置为标题的行
2. 点击 H1 或 H2 按钮
3. 或使用标题级别下拉菜单选择 H1-H6

### 列表
1. **有序列表**（1.2.3.）：
   - 点击有序列表按钮
   - 输入内容后按 Enter 自动创建下一项
   - 按两次 Enter 退出列表

2. **无序列表**（• • •）：
   - 点击无序列表按钮
   - 输入内容后按 Enter 自动创建下一项
   - 按两次 Enter 退出列表

### 缩进功能
⚠️ **重要**：缩进功能只能在列表中使用

**使用方法**：
1. 先创建一个列表（有序或无序）
2. 在列表项中，点击"增加缩进"按钮创建子列表
3. 点击"减少缩进"按钮返回上一级

**示例**：
```
• 主列表项 1
  • 子列表项 1.1
  • 子列表项 1.2
• 主列表项 2
```

### 对齐方式
选中段落或将光标放在段落中，然后点击对齐按钮：
- **左对齐**：默认对齐方式
- **居中对齐**：文字居中显示
- **右对齐**：文字靠右显示
- **两端对齐**：文字两端对齐

**提示**：点击对齐按钮后，会展开下拉菜单显示所有对齐选项

### 颜色设置
1. **字体颜色**：
   - 选中文字
   - 点击字体颜色按钮（A）
   - 选择颜色

2. **背景颜色**：
   - 选中文字
   - 点击背景颜色按钮（▮）
   - 选择颜色

### 引用和代码
- **引用块**：点击引用按钮（"），适合引用他人的话
- **代码块**：点击代码块按钮（</>），适合插入代码

### 链接
1. 选中要添加链接的文字
2. 点击链接按钮（🔗）
3. 在弹出的对话框中输入链接地址
4. 点击确定

**提示**：如果没有选中文字，会提示先选择文字

### 图片
**方法一：上传本地图片**
1. 点击图片按钮（🖼）
2. 从电脑中选择图片文件
3. 等待上传完成
4. 图片会自动插入到光标位置

**方法二：使用图片 URL**
1. 找到图片的网络地址（如：https://example.com/image.jpg）
2. 点击图片按钮
3. 在文件选择对话框中取消
4. 手动在编辑器中插入图片（需要开发者模式）

**限制**：
- 单张图片最大 5MB
- 支持所有常见图片格式（jpg, png, gif, webp 等）

## 高级技巧

### 快捷键
- **Ctrl+B**：加粗
- **Ctrl+I**：斜体
- **Ctrl+U**：下划线
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做

### 清除格式
如果文字格式混乱，可以：
1. 选中需要清除格式的文字
2. 点击清除格式按钮（🗑）
3. 所有格式会被移除，恢复为纯文本

### 复制粘贴
- **从其他地方复制**：粘贴时会保留基本格式
- **纯文本粘贴**：先粘贴，然后选中文字点击清除格式

## 常见问题

### Q1: 为什么缩进按钮点了没反应？
**A**: 缩进功能只能在列表中使用。请先创建一个有序列表或无序列表，然后在列表项中使用缩进功能。

### Q2: 对齐方式怎么选择？
**A**: 点击对齐按钮后，会展开一个下拉菜单，显示四种对齐方式：左对齐、居中、右对齐、两端对齐。鼠标悬停在每个选项上会显示具体说明。

### Q3: 图片上传失败怎么办？
**A**: 检查以下几点：
- 文件是否为图片格式
- 文件大小是否超过 5MB
- 浏览器是否允许文件访问权限

### Q4: 如何插入大量图片？
**A**: 建议：
- 将图片上传到图床（如：七牛云、阿里云 OSS）
- 获取图片 URL
- 使用图片 URL 方式插入（避免笔记内容过大）

### Q5: 链接如何修改？
**A**: 
1. 选中已有链接的文字
2. 再次点击链接按钮
3. 会显示当前链接，可以修改
4. 点击确定保存

### Q6: 如何删除链接？
**A**: 
1. 选中链接文字
2. 点击链接按钮
3. 清空链接地址
4. 点击确定

## 最佳实践

### 1. 文档结构
```
# 一级标题（H1）
## 二级标题（H2）
### 三级标题（H3）

正文内容...

- 要点 1
- 要点 2
- 要点 3
```

### 2. 代码展示
使用代码块功能：
```
function hello() {
  console.log('Hello World')
}
```

### 3. 引用他人观点
使用引用块：
> 这是一段引用的内容
> 可以跨多行

### 4. 强调重点
- 使用**加粗**强调重要内容
- 使用*斜体*表示术语或外文
- 使用颜色突出关键信息

### 5. 图片使用建议
- 截图：直接上传本地图片
- 网络图片：使用图片 URL
- 大量图片：建议使用图床服务

## 性能优化建议

1. **避免过大的图片**：单张图片建议不超过 2MB
2. **合理使用格式**：不要过度使用颜色和字体
3. **定期保存**：编辑长文档时，定期点击保存按钮
4. **分段编辑**：超长文档建议分成多篇笔记
