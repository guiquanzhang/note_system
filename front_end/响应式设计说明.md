# 📱 响应式设计说明

## 断点设置

项目使用以下断点进行响应式设计：

| 断点 | 设备类型 | 屏幕宽度 |
|------|---------|---------|
| 桌面端 | Desktop | > 768px |
| 平板端 | Tablet | ≤ 768px |
| 手机端 | Mobile | ≤ 480px |

---

## 各页面响应式适配

### 1. 营销首页 (`/`)

#### 桌面端 (> 768px)
- Hero 区域：左右双栏布局（文字 + 浮动卡片）
- 功能卡片：3 列网格布局
- 统计数据：4 列网格布局

#### 平板/手机端 (≤ 768px)
- Hero 区域：单栏布局，隐藏浮动卡片
- 功能卡片：单列布局
- 统计数据：2 列网格布局
- 标题字体缩小：56px → 36px

---

### 2. 登录/注册页 (`/login`, `/register`)

#### 桌面端 (> 768px)
- 左右分栏：品牌展示区 + 表单区
- 顶部导航栏：Logo + 首页按钮

#### 平板/手机端 (≤ 768px)
- 上下堆叠：品牌区在上，表单区在下
- 品牌区高度缩小
- 标题字体缩小：52px → 36px
- 内边距减小

---

### 3. 主布局 (`MainLayout`)

#### 桌面端 (> 768px)
- 顶部导航栏：Logo + 用户信息（头像 + 用户名 + 下拉菜单）
- 左侧边栏：固定 240px 宽度
- 右侧内容区：自适应宽度

#### 平板/手机端 (≤ 768px)
- 顶部导航栏：Logo + 用户头像（隐藏用户名）
- 左侧边栏：隐藏（可通过菜单按钮展开）
- 右侧内容区：占满全宽

#### 手机端 (≤ 480px)
- Logo 字体缩小：20px → 16px
- 图标缩小：28px → 24px

---

### 4. 笔记列表页 (`/notes`)

#### 桌面端 (> 768px)
- 页面标题 + 新建按钮：左右布局
- 笔记卡片：3 列网格布局（自适应）
- 分页器：居中显示

#### 平板/手机端 (≤ 768px)
- 页面标题 + 新建按钮：上下堆叠
- 笔记卡片：单列布局
- 内边距减小：24px → 16px
- 标题字体缩小：24px → 20px

#### 手机端 (≤ 480px)
- 内边距进一步减小：16px → 12px
- 标题字体缩小：20px → 18px

---

### 5. 笔记编辑页 (`/notes/create`, `/notes/:id/edit`)

#### 桌面端 (> 768px)
- 顶部操作栏：返回按钮 + 取消/保存按钮
- 表单区域：标准布局

#### 平板/手机端 (≤ 768px)
- 内边距减小：24px → 16px
- 按钮间距减小

#### 手机端 (≤ 480px)
- 顶部操作栏：上下堆叠
- 按钮占满宽度
- 内边距进一步减小：16px → 12px

---

## 响应式特性

### 1. 布局调整
- **桌面端**：多栏布局，充分利用宽屏空间
- **移动端**：单栏布局，垂直滚动

### 2. 字体缩放
- 标题字体根据屏幕宽度自动缩小
- 保持可读性的同时节省空间

### 3. 间距优化
- 移动端减小内边距和外边距
- 增加触摸目标的大小（按钮、链接）

### 4. 隐藏/显示元素
- 移动端隐藏次要信息（如用户名）
- 保留核心功能和内容

### 5. 导航优化
- 移动端侧边栏可折叠
- 顶部导航栏简化

## 注意事项

### 1. 触摸目标大小
- 按钮最小尺寸：44x44px（Apple 标准）
- 链接最小尺寸：48x48px（Material Design 标准）

### 2. 字体大小
- 正文最小字体：14px
- 标题最小字体：18px
- 避免使用小于 12px 的字体

### 3. 图片优化
- 使用响应式图片（srcset）
- 移动端加载小尺寸图片
- 使用 WebP 格式减小文件大小

### 4. 性能优化
- 避免使用大型背景图
- 减少动画效果（移动端）
- 使用 CSS transform 代替 position
