# AI 助手功能说明

## 📖 概述

云笔记系统集成了 AI 助手功能，帮助用户在学习和记笔记过程中获得智能问答支持。

## ✨ 功能特点

### 1. 智能问答
- 支持各种学习问题咨询
- 编程、数学、语言等多领域支持
- 上下文连续对话

### 2. 笔记辅助
- 帮助理解复杂概念
- 提供学习建议
- 解答疑难问题

### 3. 免费使用
- 使用免费 API 服务
- 零成本学习助手
- 多个 API 可选

## 🚀 快速开始

### 1. 访问 AI 助手
- 登录系统
- 点击左侧菜单 "AI 助手"

### 2. 配置 API
点击右上角"设置"按钮，选择一个 API 服务：

#### 推荐：Groq API（最快）
1. 访问：https://console.groq.com
2. 注册账号（免费）
3. 创建 API Key
4. 复制 API Key 到设置中
5. 选择模型：llama-3.1-70b-versatile

#### 备选：Cohere API
1. 访问：https://dashboard.cohere.com
2. 注册账号（免费）
3. 获取 API Key
4. 复制到设置中

#### 备选：HuggingFace API
1. 访问：https://huggingface.co/settings/tokens
2. 创建 Access Token
3. 复制到设置中

### 3. 开始对话
- 在输入框输入问题
- 点击发送或按 Enter
- AI 会给出回答

## 💡 使用技巧

### 1. 提问技巧
- 问题尽量具体明确
- 提供足够的上下文
- 可以追问细节

### 2. 快速问题
点击预设的快速问题：
- "解释一下 JavaScript 的闭包"
- "如何学习 Spring Boot？"
- "帮我理解这个概念"
- "给我一些学习建议"

### 3. 上下文对话
- AI 会记住对话历史
- 可以连续追问
- 点击"清空对话"重新开始

## 🔧 常见问题

### Q1: API Key 在哪里获取？
**A**: 
- Groq: https://console.groq.com
- Cohere: https://dashboard.cohere.com
- HuggingFace: https://huggingface.co/settings/tokens

### Q2: 为什么回答很慢？
**A**: 
- 检查网络连接
- 尝试切换其他 API
- Groq 通常最快

### Q3: 提示 API Key 无效？
**A**: 
- 检查 API Key 是否正确复制
- 确认 API Key 没有过期
- 重新生成新的 API Key

### Q4: 回答不准确怎么办？
**A**: 
- 尝试更具体的问题
- 提供更多上下文
- 换一个 API 试试

### Q5: 可以保存对话记录吗？
**A**: 
- 当前版本不保存历史
- 可以复制重要内容到笔记
- 刷新页面会清空对话

## 📝 注意事项

1. **API Key 安全**
   - 不要分享你的 API Key
   - 定期更换 API Key
   - 发现泄露立即重新生成

2. **使用限制**
   - 免费 API 有调用限制
   - 合理使用，避免滥用
   - 超限后等待重置或升级

3. **内容准确性**
   - AI 回答仅供参考
   - 重要内容需要验证
   - 不要完全依赖 AI

4. **隐私保护**
   - 不要输入敏感信息
   - 对话内容会发送到 API 服务器
   - 注意保护个人隐私
