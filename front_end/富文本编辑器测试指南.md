# 富文本编辑器测试指南

## 测试环境
- 前端地址: http://localhost:5174/
- 后端地址: http://localhost:8080/api

## 测试步骤

### 1. 启动服务
确保前后端服务都已启动：
- 后端: 运行 Spring Boot 应用
- 前端: 已在 http://localhost:5174/ 运行

### 2. 登录系统
1. 访问 http://localhost:5174/
2. 点击"登录"按钮
3. 使用已有账号登录，或先注册新账号

### 3. 测试创建笔记
1. 登录后自动跳转到笔记列表页
2. 点击"新建笔记"按钮
3. 输入笔记标题
4. 在富文本编辑器中测试以下功能：

#### 基础格式测试
- [ ] 输入文本
- [ ] 选中文本，点击"加粗"按钮
- [ ] 选中文本，点击"斜体"按钮
- [ ] 选中文本，点击"下划线"按钮
- [ ] 选中文本，点击"删除线"按钮

#### 标题测试
- [ ] 点击标题按钮，选择 H1
- [ ] 点击标题按钮，选择 H2
- [ ] 点击标题按钮，选择 H3

#### 列表测试
- [ ] 点击"有序列表"按钮，输入多行内容
- [ ] 点击"无序列表"按钮，输入多行内容

#### 颜色测试
- [ ] 选中文本，点击"字体颜色"，选择红色
- [ ] 选中文本，点击"背景颜色"，选择黄色

#### 对齐测试
- [ ] 选中段落，点击"居中对齐"
- [ ] 选中段落，点击"右对齐"
- [ ] 选中段落，点击"左对齐"

#### 其他功能测试
- [ ] 点击"引用"按钮，输入引用内容
- [ ] 点击"代码块"按钮，输入代码
- [ ] 点击"链接"按钮，插入链接
- [ ] 点击"图片"按钮，插入图片（可以使用图片 URL）

5. 选择分类（可选）
6. 输入标签（可选）
7. 点击"保存"按钮

### 4. 测试笔记列表
1. 保存后自动返回笔记列表
2. 检查新创建的笔记是否显示
3. 检查笔记预览内容是否正确（应该是纯文本，没有 HTML 标签）

### 5. 测试笔记详情
1. 点击笔记卡片
2. 进入笔记详情页
3. 检查内容是否正确渲染：
   - [ ] 标题正确显示
   - [ ] 格式正确显示（加粗、斜体等）
   - [ ] 列表正确显示
   - [ ] 颜色正确显示
   - [ ] 链接可点击
   - [ ] 图片正确显示
4. 检查元信息显示：
   - [ ] 分类标签
   - [ ] 创建时间
   - [ ] 更新时间
   - [ ] 标签列表

### 6. 测试编辑笔记
1. 在详情页点击"编辑"按钮
2. 进入编辑页面
3. 检查内容是否正确加载到编辑器
4. 修改部分内容
5. 点击"保存"按钮
6. 返回详情页，检查修改是否生效

### 7. 测试删除笔记
1. 在详情页点击"删除"按钮
2. 确认删除对话框
3. 点击"确定"
4. 检查是否返回列表页
5. 检查笔记是否已删除

### 8. 响应式测试
1. 调整浏览器窗口大小
2. 测试以下断点：
   - [ ] 桌面端（> 768px）
   - [ ] 平板端（480px - 768px）
   - [ ] 移动端（< 480px）
3. 检查编辑器工具栏是否正常显示
4. 检查按钮布局是否合理

### 9. 边界情况测试
1. 创建空内容笔记（只有标题）
2. 创建大量内容的笔记（测试性能）
3. 插入多张图片（测试内容大小）
4. 测试特殊字符输入

## 常见问题

### 1. 图片无法显示
- 检查图片 URL 是否有效
- 检查网络连接
- 如果使用本地图片，需要配置图片上传功能

### 2. 内容保存失败
- 检查后端服务是否运行
- 检查浏览器控制台错误信息
- 检查网络请求状态

### 3. 格式显示不正确
- 清除浏览器缓存
- 检查 Quill 样式是否正确加载
- 检查浏览器控制台是否有 CSS 错误

### 4. 编辑器加载慢
- 检查网络连接
- 检查是否有大量图片
- 考虑优化图片大小

## 性能建议
1. 避免在单个笔记中插入过多图片
2. 使用图片 URL 而不是 Base64 编码
3. 定期清理不需要的笔记
4. 考虑实现图片上传到服务器功能
