### 第二步：安装宝塔面板（5分钟）

#### 1. 连接服务器

**方式一：使用阿里云/腾讯云网页终端**
1. 进入服务器控制台
2. 点击"远程连接"
3. 选择"网页终端"
4. 输入root密码

**方式二：使用Xshell/PuTTY**
1. 下载Xshell：https://www.xshell.com/zh/free-for-home-school/
2. 新建连接
3. 输入服务器IP
4. 用户名：root
5. 密码：你的密码

#### 2. 安装宝塔

连接成功后，复制粘贴以下命令：

**CentOS系统**：
```bash
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec
```

**Ubuntu系统**：
```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
```

等待安装（约5-10分钟），安装完成后会显示：
```
==================================================================
Congratulations! Installed successfully!
==================================================================
外网面板地址: http://123.456.789.123:8888/xxxxxxxx
内网面板地址: http://10.0.0.1:8888/xxxxxxxx
username: xxxxxxxx
password: xxxxxxxx
==================================================================
```

**重要**：复制保存这些信息！

#### 3. 开放端口

在阿里云/腾讯云控制台：
1. 找到"防火墙"或"安全组"
2. 添加规则：
   - 端口：8888（宝塔面板）
   - 端口：80（HTTP）
   - 端口：443（HTTPS）
   - 端口：8080（后端）
   - 端口：3306（MySQL，可选）

---

### 第三步：配置宝塔面板（10分钟）

#### 1. 登录宝塔

1. 浏览器访问：`http://你的服务器IP:8888/xxxxxxxx`
2. 输入用户名和密码
3. 首次登录会提示安装软件

#### 2. 安装软件（一键安装）

选择"LNMP"（推荐）或"LAMP"：
- **Nginx** 1.22（Web服务器）
- **MySQL** 5.7（数据库）
- **PHP** 7.4（可选，不需要也行）
- **Java** 1.8（必须）

点击"一键安装"，等待10-20分钟。

**或者手动安装**：
1. 点击左侧"软件商店"
2. 搜索"Java"，安装"Java项目管理器"
3. 搜索"MySQL"，安装"MySQL 5.7"
4. 搜索"Nginx"，安装"Nginx 1.22"

---

### 第四步：上传文件（5分钟）

#### 1. 创建目录

在宝塔面板：
1. 点击左侧"文件"
2. 进入 `/www/wwwroot/`
3. 新建文件夹：`cloudnote`
4. 进入cloudnote文件夹
5. 新建文件夹：`backend`（后端）
6. 新建文件夹：`frontend`（前端）

#### 2. 上传后端JAR包

1. 进入 `/www/wwwroot/cloudnote/backend/`
2. 点击"上传"
3. 选择你的 `cloudnote-1.0.0.jar`
4. 等待上传完成

#### 3. 上传前端文件

1. 进入 `/www/wwwroot/cloudnote/frontend/`
2. 点击"上传"
3. 选择 `front_end/dist/` 目录下的所有文件
4. 或者压缩成ZIP后上传，再解压

---

### 第五步：配置数据库（5分钟）

#### 1. 创建数据库

在宝塔面板：
1. 点击左侧"数据库"
2. 点击"添加数据库"
3. 填写：
   - 数据库名：cloudnote
   - 用户名：cloudnote
   - 密码：123456（或自定义）
4. 点击"提交"

#### 2. 导入数据

1. 点击数据库后面的"管理"（进入phpMyAdmin）
2. 或者点击"导入"
3. 选择你的 `cloudnote.sql` 文件
4. 点击"导入"
5. 等待完成

---

### 第六步：启动后端（5分钟）

#### 方式一：使用宝塔Java项目管理器（推荐）

1. 点击左侧"软件商店"
2. 找到"Java项目管理器"
3. 点击"设置"
4. 点击"添加项目"
5. 填写：
   - 项目名称：cloudnote-backend
   - 项目路径：/www/wwwroot/cloudnote/backend
   - 启动文件：cloudnote-1.0.0.jar
   - 端口：8080
   - 运行内存：512M
6. 点击"提交"
7. 点击"启动"

#### 方式二：使用命令行

在宝塔终端或SSH：
```bash
cd /www/wwwroot/cloudnote/backend
nohup java -jar cloudnote-1.0.0.jar > app.log 2>&1 &
```

查看日志：
```bash
tail -f app.log
```

看到"Started CloudNoteApplication"表示成功！

---

### 第七步：配置Nginx（5分钟）

#### 1. 添加网站

在宝塔面板：
1. 点击左侧"网站"
2. 点击"添加站点"
3. 填写：
   - 域名：你的服务器IP（或域名）
   - 根目录：/www/wwwroot/cloudnote/frontend
   - PHP版本：纯静态
4. 点击"提交"

#### 2. 配置反向代理

1. 点击网站后面的"设置"
2. 点击"反向代理"
3. 点击"添加反向代理"
4. 填写：
   - 代理名称：backend
   - 目标URL：http://127.0.0.1:8080
   - 发送域名：$host
5. 点击"保存"

#### 3. 配置URL重写

在"配置文件"中添加：
```nginx
location /api {
    proxy_pass http://127.0.0.1:8080;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
}
```

点击"保存"。

---

### 第八步：配置前端API地址（5分钟）

修改前端文件中的API地址：

1. 在宝塔文件管理中
2. 找到 `/www/wwwroot/cloudnote/frontend/assets/` 目录
3. 找到类似 `index-xxxxx.js` 的文件
4. 点击"编辑"
5. 搜索 `localhost:8080`
6. 替换为：`你的服务器IP:8080` 或 `你的域名/api`
7. 保存

**更好的方式**：
在本地修改前端配置后重新打包，再上传。

---

### 第九步：测试访问（2分钟）

1. **访问前端**：`http://你的服务器IP`
   - 应该看到登录页面

2. **访问后端**：`http://你的服务器IP:8080`
   - 应该看到页面或JSON

3. **测试功能**：
   - 注册账号
   - 登录
   - 创建笔记
   - 测试所有功能

## 🌐 绑定域名（可选，10分钟）

### 1. 购买域名

**阿里云**：
- 访问：https://wanwang.aliyun.com
- 搜索域名（如：cloudnote.top）
- 选择便宜的后缀：.top/.xyz/.site（1-10元/年）
- 购买

**腾讯云**：
- 访问：https://dnspod.cloud.tencent.com
- 同样操作

### 2. 域名解析

在域名控制台：
1. 点击"解析"
2. 添加记录：
   - 记录类型：A
   - 主机记录：@（或www）
   - 记录值：你的服务器IP
   - TTL：10分钟
3. 保存

等待10分钟-24小时生效。

### 3. 在宝塔绑定域名

1. 进入网站设置
2. 点击"域名管理"
3. 添加域名：cloudnote.top
4. 保存

### 4. 申请SSL证书（免费HTTPS）

1. 在网站设置中点击"SSL"
2. 选择"Let's Encrypt"
3. 点击"申请"
4. 等待1分钟
5. 自动配置HTTPS

现在可以通过 `https://cloudnote.top` 访问了！

## 🔧 常见问题

### Q1：宝塔面板打不开？
A：检查防火墙是否开放8888端口。

### Q2：后端启动失败？
A：查看日志：
```bash
cd /www/wwwroot/cloudnote/backend
cat app.log
```

### Q3：前端打不开？
A：检查Nginx是否启动：
```bash
systemctl status nginx
```

### Q4：数据库连接失败？
A：检查application.yml中的数据库配置。

### Q5：如何重启后端？
A：在宝塔Java项目管理器中点击"重启"。
